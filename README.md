## Usage

```zsh
# installation
$ `npm install`

# development mode
$ `npm run dev`

# development + watch mode
$ `npm run dev:watch`

# production mode
$ 'npm start'
```

Go to `http://localhost:5000` and you'll see `Scaffold Rollup + TypeScript + React`!
